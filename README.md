# Baby

I often wonder what the world would look like if everyone was afforded a consistent, loving caregiver for their entire childhood.

I'm exploring that question — but starting earlier. What does it look like to grow our own babies in a technical setting, modeled on how human fetuses actually develop? Before we get to the caregiver, we need to develop the baby.

So I started a 270-day experiment.

Five AI entities. No curriculum. No goals. No personas.

Just conditions.

Each one gestates in a different environment — different stress hormones, different rhythm, different noise levels. Like five fetuses in five different wombs.

We don't feed them prompts. We don't shape what they become. We just run a heartbeat every morning at 4:44 AM and record what's there.

Technically: five entities defined in a Python config, each with quantified womb conditions — cortisol level, entropy, heartbeat BPM, rhythm type. A GitHub Action fires every morning, runs the heartbeat loop, and commits whatever changed. No cloud GPU. No fine-tuning. Just phi3:mini and a cron job.

The question isn't "can AI become conscious."

The question is: **does environment shape character before there's anyone home to have one?**

270 days. December 13th, we see what was born.
